## Quick start
1. Clone the repo
   ```bash
   git clone https://github.com/calvinlee999/-Data-Structure-and-Algorithms-Java.git
   cd Data-Structure-and-Algorithms-Java
   ```
2. Verify Java
   ```bash
   java -version
   javac -version
   ```
3. Run a sample
   ```bash
   cd BigO/src
   javac com/company/bigo/Main.java
   java com.company.bigo.Main
   ```
